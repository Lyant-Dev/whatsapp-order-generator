# WhatsApp Order Generator

A vanilla JavaScript WhatsApp Order Generator — built from scratch, no tutorials. Part of a 6-project frontend portfolio roadmap focused on learning by building.

🔗 Live Demo: [coming soon]

---

## Features

- 9 menu items with **+/−** quantity stepper
- Real-time order summary — updates as user selects items
- Auto-calculates total price using `.reduce()`
- Generates a formatted WhatsApp message with order details + customer data
- Opens WhatsApp directly with pre-filled message via `wa.me` link

---

## Tech Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript (no frameworks, no libraries)
- Remix Icon

## What I Practiced

- DOM manipulation (`querySelectorAll`, `dataset`, `innerText`, `innerHTML`)
- Array methods: `.forEach()`, `.map()`, `.filter()`, `.reduce()`

- URL encoding with `encodeURIComponent()`
- Separating data logic from render logic
